image_transformer.utils.arguments_parser module

This module provides a factory function to create an argument parser for the image-enhancer program. The parser handles command-line arguments for processing images and converting them into SVG format.

  • pixels_processors_keys (list[str]) – List of valid keys for pixel processors.

  • configurations (dict) – Dictionary containing default configuration values.
